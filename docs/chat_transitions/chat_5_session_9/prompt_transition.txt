# 📋 CONTEXTE PROJET

**Desktop-Mate** est une application hybride Python + Unity qui affiche un avatar VRM interactif sur le bureau Windows.

**Objectif final :** Assistant virtuel personnalisé avec IA conversationnelle.

# ✅ SESSIONS COMPLÉTÉES (0-9)

- Session 0 : Git Configuration
- Session 1 : Setup Python (PySide6, architecture)
- Session 2 : Unity 2022.3 LTS + URP
- Session 3 : UniVRM 0.127.3+
- Session 4 : IPC Python ↔ Unity (Socket TCP port 5555)
- Session 5 : Chargement VRM
- Session 6 : Expressions faciales (5 blendshapes)
- Session 7 : Animations & Transitions (Lerp)
- Session 8 : Clignement automatique (SmoothStep)
- Session 9 : Mouvements de tête + Réorganisation UI (3 onglets)

# 🎯 ÉTAT ACTUEL

**Interface Python (3 onglets) :**
- Connexion → Boutons connexion Unity + load VRM
- Expressions → 5 sliders expressions faciales + reset
- Animations → Auto-blink (checkbox) + Head movements (checkbox + 2 sliders freq/amp) + reset
- Options → Vitesse transition (slider) + reset

**Unity C# :**
- PythonBridge.cs (serveur IPC, 7 commandes)
- VRMLoader.cs (load/unload VRM)
- VRMBlendshapeController.cs (expressions + Lerp)
- VRMAutoBlinkController.cs (clignement SmoothStep)
- VRMHeadMovementController.cs (mouvements tête Coroutine + SmoothStep)

**Tout fonctionne :** IPC stable, VRM chargeable, expressions contrôlables, auto-blink opérationnel, head movements opérationnels.
